##Quick start commands:  
Check the status of your work

    git status
Snapshots file in preparation for versioning   
    git add [file]
example:   
    git add file.txt
add all files   
    git add .

Commit files       
    git commit -m "write your commit commentary"
    
Example:   
    git commit -m "Added reference document for workshop"
